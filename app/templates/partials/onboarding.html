First time?

You need Telegram API credentials to use this tool.

Getting Started

Step 1: Get API Credentials
  1. Go to my.telegram.org/apps
  2. Log in with your phone number
  3. Create a new application (any name/platform)
  4. Copy api_id and api_hash
Step 2: Configure

Create a .env file in the project root:

TELEGRAM_API_ID=your_api_id
TELEGRAM_API_HASH=your_api_hash

Then restart the application.

Step 3: Log In

Log in with two Telegram accounts:

  • Account A — the source (where messages are)
  • Account B — the destination (where messages go)

Rate Limiting & Safety

Built-in protections to keep your account safe:

  • Token-bucket rate limiter with jitter
  • Automatic FloodWait detection and backoff
  • Daily message cap (default: 1500/day)
  • Batch cooldowns every 20 messages
  • 2x delay for accounts < 30 days old
